# 🚀 Mission Control

**Project operations & resource planning dashboard.**

[![React](https://img.shields.io/badge/React-19-61DAFB?logo=react&logoColor=white)](https://react.dev)
[![Vite](https://img.shields.io/badge/Vite-7-646CFF?logo=vite&logoColor=white)](https://vite.dev)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-4-06B6D4?logo=tailwindcss&logoColor=white)](https://tailwindcss.com)

Multi-view visibility into teams, timelines, and risk. Built on the **Gravity** design system.

## Features

- **Project Dashboards** -- real-time status across all active projects
- **Project Tracking** -- milestones, deliverables, and progress at a glance
- **Team Capacity** -- who's available, who's overloaded, who's on fire
- **Timeline Views** -- Gantt-style planning with dependency awareness
- **Risk Assessment** -- flag blockers before they become disasters
- **Crisis Management** -- escalation workflows when things go sideways

## Quick Start

```bash
npm install
npm run dev
```

That's it. Open [http://localhost:5173](http://localhost:5173) and go.

## Build

```bash
npm run build
npm run preview
```
